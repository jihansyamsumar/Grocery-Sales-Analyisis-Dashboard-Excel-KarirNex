# Grocery-Sales-Analyisis-Dashboard-Excel-KarirNex
This project contains an analysis of basic foodstuff sales data using Microsoft Excel. Data processing is carried out using Excel formulas, pivot tables, and dashboard visualisations to generate insights such as city performance, sales platforms, discount categories, and customer purchasing behaviour.

## This project aims to train the following skills:
1. Data cleaning & analysis
2. Use of formulas (SUMIFS, AVERAGEIFS, COUNTIFS, etc.)
3. Creating pivot tables & pivot charts
4. Developing informative dashboards for decision making
